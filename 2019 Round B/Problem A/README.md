# A. Building Palindromes

### 描述

给定一行的N个方块，每个方块有一个从A到Z的字母在上面。方块的序列从左到右分别为1到N。定义回文是一个从前向和后向读起来都是一样的字符串，例如ANNA、RACECAR、AAA等。

给定Q个问题，每个问题存在一个范围[Li，Ri]，问该范围内的字母能否构成一个回文。输出回答为正的问题数目。

### 分析

**回文判断条件**：要判断一个子字符串是否为回文，我们只需计算其包含的所有字母的频率。如果频率为奇数次的字母的出现数目大于1，则不能构成回文。

对于小数据集，我们可以对每一个查询都扫描一遍子字符串，计算其字母频率是否符合要求。时间复杂度为O(N×Q).

对于大数据集，我们预先对整个字符串序列计算频率的前缀和。接着，对于每个查询字串，我们可以检查所有字符的奇偶性。最终的时间复杂度为O((N + Q) × |character set|)。
